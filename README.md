# Dayflow - Human Resource Management System

> Every workday, perfectly aligned.

A comprehensive HRMS built for the Odoo Hackathon 2025.

## Features

- **Authentication** - Sign Up/Sign In with role-based access (Admin/Employee)
- **Employee Dashboard** - Profile, Attendance, Leave Requests
- **Admin Dashboard** - Employee management, Attendance records, Leave approvals
- **Attendance Tracking** - Check-in/Check-out, Daily/Weekly views
- **Leave Management** - Apply for leave, Approval workflow
- **Payroll** - View salary details

## Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js + Express.js
- **Database**: MongoDB
- **Authentication**: JWT

## Getting Started

```bash
# Install dependencies
npm install

# Run backend
cd backend && npm run dev

# Run frontend
cd frontend && npm run dev
```
